第1回 練習問題：Eclipseの操作

Eclipse でプロジェクトとクラスを作成し、プログラムを実行する手順を確認する問題です。次の3ステップで進めてください。

ステップ一覧

ステップ 内容 確認
1 新しい Java プロジェクトを作成する パッケージ・エクスプローラーにプロジェクトが表示される
2 main メソッドを持つクラスを作成し、「Hello」と表示して実行する コンソールに「Hello」と表示される
3 表示する文字列を変えたり、複数行の println を試す 保存すると自動コンパイルされ、実行で結果が変わることを確認する

【ステップ1】プロジェクトの作成

Eclipse で新しい Java プロジェクトを作成してください。プロジェクト名は自由（例：Lesson1Practice）で構いません。

ヒント

メニュー ファイル → 新規 → Java プロジェクト を選び、プロジェクト名を入力して「完了」をクリックします。左側のパッケージ・エクスプローラーにプロジェクトが表示されれば成功です。

手順の目安
  1. ファイル → 新規 → Java プロジェクト
  2. プロジェクト名を入力（例：Lesson1Practice）
  3. 「完了」をクリック
  4. パッケージ・エクスプローラーにプロジェクトが表示されていることを確認

【ステップ2】クラスの作成と実行

作成したプロジェクトの中に、main メソッドを持つクラスを作成し、System.out.println("Hello"); で「Hello」と表示して実行してください。

ヒント

プロジェクトを右クリック → 新規 → クラス。クラス名を入力し、「public static void main(String[] args) にチェックを入れる」にチェックを入れて「完了」。main の中に System.out.println("Hello"); を書き、保存してから緑の三角（実行）をクリックします。

解答例（コード） 
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}

【ステップ3】表示内容を変えて試す

次のいずれか（または両方）を試してください。

  • 表示する文字列を「Hello」から別の文字（例：「こんにちは」）に変えて保存し、再実行する。
  • println を2行以上書き、複数行がコンソールに表示されることを確認する。

保存すると自動でコンパイルされ、実行ボタンで新しい結果が表示されることを確認しましょう。

ヒント

コードを変更したら必ず保存（Ctrl+S / Command+S）してから実行します。複数行表示する例：System.out.println("1行目"); の下に System.out.println("2行目"); を追加する。

解答例（複数行表示） 
public class Hello {
    public static void main(String[] args) {
        System.out.println("1行目");
        System.out.println("2行目");
    }
}

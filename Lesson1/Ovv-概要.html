第1回：Eclipseの操作 ～環境に慣れる～

1. 必要な前提知識

  • 特にありません。パソコンの基本操作（フォルダ・ファイルの概念）がわかっていれば大丈夫です。
  • JDK： Java のプログラムを動かすために必要なソフトウェア。Eclipse の前にインストールします。
  • Eclipse： Java を書いて実行するための統合開発環境（IDE）です。

2. ポイント解説

💡 ワークスペースは「作業場」

Eclipse を起動すると、最初に「ワークスペース」というフォルダを選びます。ここにプロジェクトや設定が保存されます。同じワークスペースを指定すれば、前回の続きから作業できます。

💡 プロジェクトとクラスの作成・実行

Java のプログラムは「プロジェクト」の中に「クラス」を作って書きます。Eclipse では「新規 Java プロジェクト」でプロジェクトを作り、「新規クラス」で main メソッドを持つクラスを作成。実行ボタン（緑の三角）でプログラムを動かします。

  • 自動コンパイル： 保存すると自動でコンパイルされ、エラーがあれば一覧に表示されます。
  • JDK 設定： プロジェクトで使う Java のバージョン（JDK）を正しく設定しておきましょう。

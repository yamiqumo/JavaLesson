基本問題① 動物クラスを利用した問題（継承とオーバーライド）

動物クラス Animal を土台に、継承とオーバーライドを使って次の3段階で拡張してください。

フィールド／メソッド一覧（Animal）

区分 名前 型 説明 登場する段階
フィールド name String 動物の名前 ステップ1〜3
メソッド cry() void 鳴き声を表示（Animal では「〇〇が鳴きます。」） ステップ1〜3

ステップ一覧

ステップ 内容 確認
1 Animal クラスを作る（name、コンストラクタ Animal(String name)、cry() で「〇〇が鳴きます。」） Main で new Animal("何か") を生成し、cry() を呼ぶ
2 Dog が Animal を継承。cry() をオーバーライドして「ワンワン」と表示 Main で new Dog("ポチ") を生成し、cry() で「ワンワン」と出ることを確認
3 Cat が Animal を継承。cry() をオーバーライドして「ニャー」と表示 Main で Dog と Cat を生成し、それぞれ cry() で「ワンワン」「ニャー」と出ることを確認
【ステップ1】Animal クラスを作る
上記「フィールド／メソッド一覧（Animal）」と「ステップ一覧」に従い、Animal クラスを作成してください。Main で new Animal("何か") を生成し、cry() を呼んで確認してください。
ヒント

cry() では System.out.println(name + "が鳴きます。"); のように表示します。

解答例 
public class Animal {
    private String name;
    public Animal(String name) { this.name = name; }
    public void cry() {
        System.out.println(name + "が鳴きます。");
    }
}


【ステップ2】Dog クラスで Animal を継承し、cry() をオーバーライドする
Animal を継承した Dog クラスを作成してください。
・extends Animal で継承する。コンストラクタで super(name) を呼ぶ。
・cry() をオーバーライドし、「ワンワン」と表示する。
・Main で new Dog("ポチ") を生成し、cry() を呼ぶと「ワンワン」と出ることを確認する。
ヒント

クラス宣言は public class Dog extends Animal。オーバーライドした cry() では System.out.println("ワンワン"); とします。

解答例 
public class Dog extends Animal {
    public Dog(String name) { super(name); }
    @Override
    public void cry() {
        System.out.println("ワンワン");
    }
}


【ステップ3】Cat クラスも継承し、Main で Dog と Cat の両方を試す
Animal を継承した Cat クラスを作成してください。
・cry() をオーバーライドし、「ニャー」と表示する。
・Main で new Dog("ポチ") と new Cat("タマ") を生成し、それぞれ cry() を呼ぶと「ワンワン」「ニャー」と出ることを確認する。
ヒント

Cat は Dog と同様に extends Animal で継承し、cry() で「ニャー」と表示します。

解答例 
public class Cat extends Animal {
    public Cat(String name) { super(name); }
    @Override
    public void cry() {
        System.out.println("ニャー");
    }
}

// Main
Dog d = new Dog("ポチ");
Cat c = new Cat("タマ");
d.cry();  // ワンワン
c.cry();  // ニャー

基本問題② 社員クラスを利用した問題（継承とオーバーライド）

社員クラス Employee を土台に、継承とオーバーライドを使って次の3段階で拡張してください。

フィールド／メソッド一覧（Employee）

区分 名前 型 説明 登場する段階
フィールド name String 社員の名前 ステップ1〜3
フィールド department String 部署名 ステップ1〜3
メソッド getName() String 名前を返す getter（サブクラス用） ステップ1〜3
メソッド getDepartment() String 部署を返す getter（サブクラス用） ステップ1〜3
メソッド introduce() void 自己紹介を表示（Employee では「〇〇です。〇〇部です。」） ステップ1〜3

ステップ一覧

ステップ 内容 確認
1 Employee クラスを作る（name, department、コンストラクタ、getName/getDepartment、introduce() で「〇〇です。〇〇部です。」） Main で new Employee("山田", "営業") を生成し、introduce() を呼ぶ
2 Manager が Employee を継承。introduce() をオーバーライドして「〇〇です。〇〇部の管理職です。」 Main で new Manager("佐藤", "開発") を生成し、introduce() を呼ぶ
3 PartTimer が Employee を継承。introduce() をオーバーライドして「〇〇です。パートで勤務しています。」 Main で Manager と PartTimer を生成し、それぞれ introduce() を呼んで確認
【ステップ1】Employee クラスを作る
上記「フィールド／メソッド一覧（Employee）」と「ステップ一覧」に従い、Employee クラスを作成してください。Main で new Employee("山田", "営業") を生成し、introduce() を呼んで確認してください。
ヒント

introduce() では name と department を使って自己紹介文を表示します。getter は return name; と return department; で返します。

解答例 
public class Employee {
    private String name;
    private String department;
    public Employee(String name, String department) {
        this.name = name;
        this.department = department;
    }
    public String getName() { return name; }
    public String getDepartment() { return department; }
    public void introduce() {
        System.out.println(name + "です。" + department + "部です。");
    }
}


【ステップ2】Manager クラスで Employee を継承し、introduce() をオーバーライドする
Employee を継承した Manager クラスを作成してください。
・extends Employee で継承する。コンストラクタ Manager(String name, String department) で super(name, department) を呼ぶ。
・introduce() をオーバーライドし、「〇〇です。〇〇部の管理職です。」と表示する。
・Main で new Manager("佐藤", "開発") を生成し、introduce() を呼んで確認する。
ヒント

Manager では introduce() をオーバーライドし、Employee の getName() と getDepartment() を使って「〇〇部の管理職です。」と表示します。

解答例 
public class Manager extends Employee {
    public Manager(String name, String department) { super(name, department); }
    @Override
    public void introduce() {
        System.out.println(getName() + "です。" + getDepartment() + "部の管理職です。");
    }
}


【ステップ3】PartTimer クラスも継承し、Main で Manager と PartTimer の両方を試す
Employee を継承した PartTimer クラスを作成してください。
・introduce() をオーバーライドし、「〇〇です。パートで勤務しています。」と表示する。
・Main で new Manager("佐藤", "開発") と new PartTimer("鈴木", "総務") を生成し、それぞれ introduce() を呼んで確認する。

PartTimer は Manager と同様に extends Employee で継承し、introduce() で「〇〇です。パートで勤務しています。」と表示します。

解答例 
public class PartTimer extends Employee {
    public PartTimer(String name, String department) { super(name, department); }

    }

発展問題③ 繰り返しや配列などを使った少し高度な問題

【問題】Animal のサブクラスを配列で持ち、for で cry() を呼ぶ

基本問題①で作った Animal・Dog・Cat を使います。

項目 内容
配列 Animal[] 型の配列を用意し、Dog と Cat のインスタンスを2つずつ入れる（例： ポチ、タマ、ハチ、ミケ）
繰り返し for 文（または拡張 for 文）で配列を回し、各要素の cry() を呼ぶ
確認 実行すると「ワンワン」「ニャー」「ワンワン」「ニャー」の順で表示されることを確認する
ヒント

Animal 型の変数に Dog や Cat のインスタンスを入れることができます（スーパークラス型でサブクラスのインスタンスを参照）。配列は Animal[] animals = { new Dog("ポチ"), new Cat("タマ"), new Dog("ハチ"), new Cat("ミケ") }; のように作れます。

解答例 
Animal[] animals = {
    new Dog("ポチ"),
    new Cat("タマ"),
    new Dog("ハチ"),
    new Cat("ミケ")
};
for (Animal a : animals) {
    a.cry();
}

【練習問題】 第11回 メンバの可視性（アクセス修飾子）＋復習

Java研修 練習問題集 — 第11回 アクセス修飾子（復習：コンストラクタ・継承）

この回では、コンストラクタとカプセル化（Lesson9）・継承（Lesson10）の復習をしつつ、アクセス修飾子（public / private / protected）を意識して、動物クラス（Animal）と社員クラス（Employee）を扱います。他レッスンの練習問題と同じ題材で一貫させています。

基本問題① 動物クラスを利用した問題（アクセス修飾子＋継承＋protected）

動物クラス Animal を、コンストラクタ・カプセル化・アクセス修飾子を付けたうえで作成し、protected メソッドを追加してから、Dog・Cat で継承とオーバーライドの復習をします。

フィールド／メソッド一覧（Animal）

区分 名前 型 説明 修飾子の目安
フィールド name String 動物の名前 private
コンストラクタ Animal(String name) — 名前を初期化 public
メソッド getName() String 名前を返す getter public
メソッド cry() void 鳴き声を表示（Animal では「〇〇が鳴きます。」） public
メソッド printCry(String message) void 引数の文字列をそのまま表示（サブクラスからだけ使いたい） protected

Dog・Cat（Lesson10 と同様）

  • Dog — extends Animal。コンストラクタで super(name)。cry() をオーバーライドして「ワンワン」と表示（printCry("ワンワン") を呼ぶと protected の利用になる）。
  • Cat — 同様に extends Animal。cry() をオーバーライドして「ニャー」と表示。
【ステップ1】Animal：コンストラクタ・カプセル化・アクセス修飾子（復習）
Animal クラスを作成してください。
・フィールド name（String）は private、コンストラクタ Animal(String name) は public、getter getName() は public。
・public の cry() で「〇〇が鳴きます。」と表示する（Lesson10 の Animal と同じ仕様）。
ヒント

コンストラクタで this.name = name;。cry() では System.out.println(name + "が鳴きます。"); でよいです。

解答例 
public class Animal {
    private String name;

    public Animal(String name) {
        this.name = name;
    }

    public String getName() { return name; }

    public void cry() {
        System.out.println(name + "が鳴きます。");
    }
}


【ステップ2】Animal に protected メソッドを追加する
Animal に、protected のメソッド printCry(String message) を追加してください。中身は System.out.println(message); だけでよいです。
・cry() の表示を、printCry(name + "が鳴きます。"); を呼ぶ形に書き換えてください。これにより、サブクラスからも printCry が使えます。
ヒント

protected void printCry(String message) { System.out.println(message); } とし、cry() 内で printCry(name + "が鳴きます。"); を呼びます。

解答例 
// cry() を printCry を使う形に変更
public void cry() {
    printCry(name + "が鳴きます。");
}

// サブクラスからだけ使えるように protected
protected void printCry(String message) {
    System.out.println(message);
}


【ステップ3】Dog・Cat で継承とオーバーライドの復習＋protected の利用
Dog クラスで extends Animal し、コンストラクタで super(name) を呼んでください。cry() をオーバーライドし、printCry("ワンワン"); を呼んで「ワンワン」と表示する（protected なのでサブクラスから呼べる）。
同様に Cat クラスで extends Animal し、cry() で printCry("ニャー"); を呼んでください。
ヒント

Dog は public class Dog extends Animal、コンストラクタ public Dog(String name) { super(name); }。cry() は @Override を付けて printCry("ワンワン"); のみ。

解答例 
public class Dog extends Animal {
    public Dog(String name) { super(name); }
    @Override
    public void cry() {
        printCry("ワンワン");  // protected なのでサブクラスから呼べる
    }
}

public class Cat extends Animal {
    public Cat(String name) { super(name); }
    @Override
    public void cry() {
        printCry("ニャー");
    }
}


【ステップ4】Main で Animal・Dog・Cat を確認する
Main の main で、new Animal("何か")、new Dog("ポチ")、new Cat("タマ") を生成し、それぞれ cry() を呼んで「何かが鳴きます。」「ワンワン」「ニャー」と出ることを確認してください。
解答例 
Animal a = new Animal("何か");
Dog d = new Dog("ポチ");
Cat c = new Cat("タマ");
a.cry();  // 何かが鳴きます。
d.cry();  // ワンワン
c.cry();  // ニャー

基本問題② 社員クラスを利用した問題（アクセス修飾子＋継承＋protected）

社員クラス Employee を、コンストラクタ・カプセル化・アクセス修飾子を付けたうえで作成し、protected メソッドを追加してから、Manager・PartTimer で継承とオーバーライドの復習をします（Lesson10・12 の社員仕様に合わせています）。

フィールド／メソッド一覧（Employee）

区分 名前 型 説明 修飾子の目安
フィールド name, department String 社員の名前・部署名 private
コンストラクタ Employee(String name, String department) — 名前・部署を初期化 public
メソッド getName(), getDepartment() String getter（サブクラス用） public
メソッド introduce() void 自己紹介を表示（Employee では「〇〇です。〇〇部です。」） public
メソッド getIntroductionSuffix() String 自己紹介の末尾部分を返す（サブクラスでオーバーライドして変えたい） protected

Manager・PartTimer（Lesson10・12 と同様）

  • Manager — extends Employee。コンストラクタで super(name, department)。getIntroductionSuffix() をオーバーライドして「部の管理職です。」を返す。これにより、introduce() を継承のままにすると「〇〇です。〇〇部の管理職です。」と表示される。
  • PartTimer — extends Employee。introduce() をオーバーライドして「〇〇です。パートで勤務しています。」と表示。
【ステップ1】Employee：コンストラクタ・カプセル化・アクセス修飾子（復習）
Employee クラスを作成してください。
・フィールド name と department（String）は private、コンストラクタ Employee(String name, String department) は public、getter は public。
・public の introduce() で「〇〇です。〇〇部です。」と表示する（Lesson10・12 と同じ仕様）。
ヒント

introduce() では System.out.println(name + "です。" + department + "部です。"); でよいです（ステップ2で getIntroductionSuffix() を使う形にします）。

解答例 
public class Employee {
    private String name;
    private String department;

    public Employee(String name, String department) {
        this.name = name;
        this.department = department;
    }

    public String getName() { return name; }
    public String getDepartment() { return department; }

    public void introduce() {
        System.out.println(name + "です。" + department + "部です。");
    }
}


【ステップ2】Employee に protected メソッドを追加する
Employee に、protected のメソッド getIntroductionSuffix()（戻り値 String）を追加してください。Employee では「部です。」を返すようにします。
・introduce() の表示を、getName() + "です。" + getDepartment() + getIntroductionSuffix() を使う形に書き換えてください。これにより、サブクラスで getIntroductionSuffix() をオーバーライドすれば、紹介文の末尾だけ変えられます。
ヒント

protected String getIntroductionSuffix() { return "部です。"; } とし、introduce() では System.out.println(getName() + "です。" + getDepartment() + getIntroductionSuffix()); とします。

解答例 
// introduce() を getIntroductionSuffix を使う形に変更
public void introduce() {
    System.out.println(getName() + "です。" + getDepartment() + getIntroductionSuffix());
}

// サブクラスでオーバーライドして末尾を変えられるように protected
protected String getIntroductionSuffix() {
    return "部です。";
}


【ステップ3】Manager・PartTimer で継承とオーバーライドの復習＋protected の利用
Manager クラスで extends Employee し、コンストラクタ Manager(String name, String department) で super(name, department) を呼んでください。getIntroductionSuffix() をオーバーライドして「部の管理職です。」を返す。introduce() は継承のままなので、「〇〇です。〇〇部の管理職です。」と表示されます。
PartTimer クラスで extends Employee し、introduce() をオーバーライドして「〇〇です。パートで勤務しています。」と表示する（Lesson10・12 と同じ）。
ヒント

Manager は getIntroductionSuffix() だけオーバーライド。PartTimer は introduce() をオーバーライドし、getName() を使って「〇〇です。パートで勤務しています。」と表示します。

解答例 
public class Manager extends Employee {
    public Manager(String name, String department) { super(name, department); }
    @Override
    protected String getIntroductionSuffix() {
        return "部の管理職です。";
    }
}

public class PartTimer extends Employee {
    public PartTimer(String name, String department) { super(name, department); }
    @Override
    public void introduce() {
        System.out.println(getName() + "です。パートで勤務しています。");
    }
}


【ステップ4】Main で Employee・Manager・PartTimer を確認する
Main の main で、new Employee("山田", "営業")、new Manager("佐藤", "開発")、new PartTimer("鈴木", "総務") を生成し、それぞれ introduce() を呼んで「山田です。営業部です。」「佐藤です。開発部の管理職です。」「鈴木です。パートで勤務しています。」と出ることを確認してください。
解答例 
Employee e = new Employee("山田", "営業");
Manager m = new Manager("佐藤", "開発");
PartTimer p = new PartTimer("鈴木", "総務");
e.introduce();  // 山田です。営業部です。
m.introduce();  // 佐藤です。開発部の管理職です。
p.introduce();  // 鈴木です。パートで勤務しています。

確認問題 アクセス修飾子の整理

次の説明に当てはまるアクセス修飾子を選んでください（public / private / protected）。

  1. 「どこからでも」アクセスできる。
  2. 「そのクラスの中だけ」からアクセスできる。
  3. 「同じクラスとサブクラス」からアクセスできる。
解答

① public　② private　③ protected

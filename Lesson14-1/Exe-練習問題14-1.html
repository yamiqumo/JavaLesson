基本問題① 動物クラス — Cryable 型とポリモーフィズム

【問題1】 Cryable 型の変数を1つ用意し、そこに new Dog("ポチ") を代入して cry() を呼んでください。続けて、同じ変数に new Cat("タマ") を代入し、もう一度 cry() を呼んでください。同じ変数・同じメソッドで、犬と猫それぞれの鳴き方が出ることを確認します。

ステップ内容
1Cryable c = new Dog("ポチ"); で変数に犬を代入し、c.cry(); を呼ぶ。
2同じ変数に c = new Cat("タマ"); を代入し、再度 c.cry(); を呼ぶ。
ヒント

Cryable 型の変数には、Cryable を実装した Dog も Cat も代入できます。代入したインスタンスに応じて、cry() の動きが変わります（ポリモーフィズム）。

解答例

【解説・確認用】Cryable 型の変数に Dog / Cat を代入し、同じ cry() で異なる表示になることを確認します。

// インスタンス化（Cryable 型の変数に Dog を代入）
Cryable c = new Dog("ポチ");
c.cry();
// 同じ変数に Cat を代入して cry()（ポリモーフィズム）
c = new Cat("タマ");
c.cry();

【問題2】 Cryable 型の配列を用意し、その中に Dog と Cat のインスタンスを複数入れます。for 文（添え字を使う形）で配列を回し、それぞれ cry() を呼んでください。

ステップ内容
1Cryable[] arr = new Cryable[3]; で配列を用意する。
2各要素に new Dog(...) や new Cat(...) を代入する。
3for (int i = 0; i < arr.length; i++) で arr[i].cry(); を呼ぶ。
ヒント

Cryable 型の配列なら、要素ごとに Dog や Cat を入れてよいです。for (int i = 0; i < arr.length; i++) で回し、arr[i].cry() を呼ぶと、その要素に応じてワンワン・ニャーと表示されます。

解答例

【解説・確認用】Cryable 型の配列に Dog と Cat を入れ、for で cry() を呼びます。

// インスタンス化（Cryable 型の配列に Dog / Cat を代入）
Cryable[] arr = new Cryable[3];
arr[0] = new Dog("ポチ");
arr[1] = new Cat("タマ");
arr[2] = new Dog("シロ");
// for で配列を回して cry()（各要素の型に応じて鳴き方が変わる）
for (int i = 0; i < arr.length; i++) {
    arr[i].cry();
}

【問題3】 問題2と同じ配列を用意したうえで、今度は拡張 for 文（for (Cryable c : arr)）で配列を回し、各要素の cry() を呼んでください。

ステップ内容
1Cryable 型の配列に Dog / Cat を入れる（問題2と同様）。
2for (Cryable c : arr) { c.cry(); } で回す。
ヒント

拡張 for では「配列の要素を1つずつ変数 c に取り出す」ので、c の型は Cryable にします。c.cry() でその要素の cry() が呼ばれます。

解答例

【解説・確認用】拡張 for で Cryable 型の配列を回し、cry() を呼びます。

// インスタンス化（Cryable 型の配列に Dog / Cat を代入）
Cryable[] arr = new Cryable[3];
arr[0] = new Dog("ポチ");
arr[1] = new Cat("タマ");
arr[2] = new Dog("シロ");
// 拡張 for で配列を回して cry()
for (Cryable c : arr) {
    c.cry();
}

基本問題② 社員クラス — Payable 型とポリモーフィズム

【問題4】 Payable 型の変数を1つ用意し、そこに new Manager("田中", 300000, 50000) を代入して calcSalary() の戻り値を表示してください。続けて、同じ変数に new PartTimer("山田", 20, 1500) を代入し、再度 calcSalary() の戻り値を表示してください。

ステップ内容
1Payable 型の変数に Manager を代入し、calcSalary() の戻り値を int で受け、表示する。
2同じ変数に PartTimer を代入し、再度 calcSalary() の戻り値を表示する。
ヒント

Payable 型の変数には Manager も PartTimer も代入できます。戻り値は一度変数に代入してから return するルールに従い、呼び出し側でも int sal = p.calcSalary(); のように受け取って表示します。

解答例

【解説・確認用】Payable 型の変数に Manager / PartTimer を代入し、同じ calcSalary() で異なる計算結果になることを確認します。

// インスタンス化（Payable 型の変数に Manager を代入）
Payable p = new Manager("田中", 300000, 50000);
int sal = p.calcSalary();
System.out.println("給与: " + sal);
// 同じ変数に PartTimer を代入して calcSalary()（ポリモーフィズム）
p = new PartTimer("山田", 20, 1500);
sal = p.calcSalary();
System.out.println("給与: " + sal);

【問題5】 Payable 型の配列を用意し、その中に Manager と PartTimer のインスタンスを複数入れます。for 文で配列を回し、それぞれ calcSalary() を呼び、結果を表示してください。

ステップ内容
1Payable[] arr = new Payable[2]; などで配列を用意する。
2各要素に new Manager(...) や new PartTimer(...) を代入する。
3for で配列を回し、calcSalary() の戻り値を表示する。
ヒント

Payable 型の配列に Manager と PartTimer を入れ、for で arr[i].calcSalary() を呼び、戻り値を System.out.println で表示します。

解答例

【解説・確認用】Payable 型の配列で for し、給与を表示します。

// インスタンス化（Payable 型の配列に Manager / PartTimer を代入）
Payable[] arr = new Payable[2];
arr[0] = new Manager("田中", 300000, 50000);
arr[1] = new PartTimer("山田", 20, 1500);
// for で配列を回して calcSalary() の戻り値を表示
for (int i = 0; i < arr.length; i++) {
    int sal = arr[i].calcSalary();
    System.out.println("給与: " + sal);
}

【問題6】 問題5と同じ配列を用意したうえで、for で配列を回しながら calcSalary() の戻り値を合計し、最後に「給与合計: ○○円」と表示してください。

ステップ内容
1Payable 型の配列に Manager / PartTimer を入れる（問題5と同様）。
2int total = 0 を用意し、for で各要素の calcSalary() を加算する。
3ループ後に total を表示する。
ヒント

合計用の変数 int total = 0 を用意し、for の中で total += arr[i].calcSalary(); のように加算します。ポリモーフィズムのまま、それぞれの計算結果が足されていきます。

解答例

【解説・確認用】Payable 型の配列を回し、給与を合計して表示します。

// インスタンス化（Payable 型の配列に Manager / PartTimer を代入）
Payable[] arr = new Payable[2];
arr[0] = new Manager("田中", 300000, 50000);
arr[1] = new PartTimer("山田", 20, 1500);
int total = 0;
// for で配列を回し、calcSalary() の戻り値を合計
for (int i = 0; i < arr.length; i++) {
    int sal = arr[i].calcSalary();
    total = total + sal;
}
System.out.println("給与合計: " + total + "円");

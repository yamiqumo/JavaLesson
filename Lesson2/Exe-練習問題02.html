第2回 練習問題：変数・型・リテラル

変数宣言、リテラル、print/println を使った問題です。次の3ステップで進めてください。

ステップ一覧

ステップ 内容 確認
1 整数・小数・文字列のリテラルを println で表示する コンソールに数値と文字列が正しく表示される
2 int / double / String の変数を宣言し、代入して表示する 変数に代入した値がコンソールに表示される
3 print と println を組み合わせて、1行に複数項目を表示する 改行の位置が意図どおりになる

【ステップ1】リテラルの表示

main メソッド内で、次の3つを System.out.println を使って表示してください。

  • 整数のリテラル（例：100）
  • 小数のリテラル（例：3.14）
  • 文字列のリテラル（例："Hello"）
ヒント

整数はそのまま println(100)、小数は println(3.14)、文字列はダブルクォートで println("Hello") と書きます。

解答例 
System.out.println(100);   // 整数リテラル
System.out.println(3.14);   // 小数リテラル
System.out.println("Hello"); // 文字列リテラル

【ステップ2】変数の宣言と表示

次の3つの変数を宣言し、値を代入したうえで、それぞれ println で表示してください。

変数名 型 代入する値の例
ageint20
heightdouble1.65
nameString"太郎"
ヒント

宣言と代入は int age = 20; のように「型 変数名 = 値;」で書きます。表示は println(age) のように変数名を渡します。

解答例 
// 変数を宣言し、値を代入
int age = 20;
double height = 1.65;
String name = "太郎";
// コンソールに表示
System.out.println(age);
System.out.println(height);
System.out.println(name);

【ステップ3】print と println の組み合わせ

print と println を使って、「名前：太郎 年齢：20」のように、ラベルと値を1行で表示するプログラムを書いてください。2つ以上の項目を print で続けて表示し、最後に println で改行する形にします。

ヒント

print は改行しないので、print("名前："); print("太郎 "); print("年齢："); println(20); のようにすると1行にまとまります。

解答例 
// print は改行しない。最後の println で改行
System.out.print("名前：");
System.out.print("太郎 ");
System.out.print("年齢：");
System.out.println(20);

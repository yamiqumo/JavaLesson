第2回：基本文法(1) ～変数・型・リテラル～

1. 必要な前提知識

この回は、一つずつ順に進めれば、プログラミングに慣れていない方でも理解できる内容です。

  • Eclipse の操作： プロジェクトとクラスを作成し、main メソッドでプログラムを実行できること。
  • ソースコード： Java のプログラムはクラスの中にメソッドを書いた「ソースコード」で表現されます。
  • print / println： コンソールに文字などを表示するためのメソッド（第1回で触れた System.out.println）。

2. ポイント解説

💡 変数と型は「データを入れる箱と種類」

変数はデータを入れておく「箱」です。Java では変数に型（int, double, String など）を指定し、「どんな種類の値が入るか」を決めます。型に合った値だけを代入できます。

  • int： 整数（例：10、-3）。
  • double： 小数（例：3.14）。
  • String： 文字列（例："Hello"）。

💡 リテラルは「プログラムに直接書く値」

リテラルは、ソースコードにそのまま書く値です。整数リテラル（100）、小数リテラル（1.5）、文字列リテラル（"こんにちは"）などがあります。型とリテラルを組み合わせて変数に代入したり、println で表示したりします。

💡 コメントとコードの整形

コメント（// や /* ... */）はプログラムとして実行されません。説明やメモを残すために使います。インデント（字下げ）や改行でコードを整えると読みやすくなります。

【確認】 boolean の理解を確認

Part 1：boolean の理解を確認（約20〜25分）

boolean とは

boolean は「真（true）か偽（false）か」だけを表す型です。条件分岐（if 文）や繰り返し（for / while）の条件式は、この boolean 型の値になります。

  • リテラル：true と false のみ。
  • 関係演算子（==, !=, <, >, <=, >=）の結果は boolean。
  • 論理演算子（&& かつ、|| または、! 否定）も boolean を返す。

【確認1】次の式の結果は true か false か

変数は int a = 10; int b = 3; とします。次の条件式の結果（true / false）を答えてください。

No. 式 答え
1a > 5（穴埋め）
2b == 3（穴埋め）
3a <= 10（穴埋め）
4a != b（穴埋め）
5a > 10（穴埋め）
答え

1. true（10 は 5 より大きい）
2. true（b は 3）
3. true（10 は 10 以下）
4. true（10 と 3 は等しくない）
5. false（10 は 10 より大きくない）

【確認2】論理演算子 && と ||

&& は「かつ」、|| は「または」です。int x = 5; int y = 8; のとき、次の式の結果を答えてください。

No. 式 答え
1x > 0 && y > 0（穴埋め）
2x > 10 || y > 5（穴埋め）
3x > 10 && y > 10（穴埋め）
答え

1. true（x>0 も y>0 も true）
2. true（y>5 が true なので全体が true）
3. false（x>10 が false なので && の結果は false）

【確認3】if の条件になる式

次のコードを実行したとき、どのメッセージが表示されますか。int n = 7; とします。

if (n >= 10) {
    System.out.println("A");
} else if (n >= 5) {
    System.out.println("B");
} else {
    System.out.println("C");
}

表示されるのは A / B / C のどれですか。

答え

B が表示されます。n=7 なので n>=10 は false、n>=5 は true のため else if のブロックが実行されます。

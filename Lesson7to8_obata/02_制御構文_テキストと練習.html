【テキスト＋練習】 制御構文（if / switch / for / while）

Part 2：制御構文（約45〜50分）

1. このパートの目標

  • if / else if / else で条件分岐を書ける。
  • switch 文で値に応じた分岐を書ける。
  • for 文と while 文で繰り返しを書ける。

2. if 文（条件分岐）

条件が true のときだけブロック内を実行します。else if で別の条件を、else で「どれにも当てはまらないとき」を書けます。

int score = 75;
if (score >= 80) {
    System.out.println("優");
} else if (score >= 60) {
    System.out.println("良");
} else {
    System.out.println("要努力");
}

3. switch 文

ある変数や式の値に応じて、case で分岐します。break を忘れると次の case に落ちるので注意。

int day = 2;
switch (day) {
    case 1: System.out.println("月"); break;
    case 2: System.out.println("火"); break;
    case 3: System.out.println("水"); break;
    default: System.out.println("その他"); break;
}

4. for 文（繰り返し）

「初期化 → 条件が true の間ブロック実行 → 更新」を繰り返します。

for (int i = 0; i < 3; i++) {
    System.out.println("i = " + i);
}
// 0, 1, 2 が表示される

5. while 文（繰り返し）

条件が true の間、ブロックを繰り返します。条件がずっと true だと無限ループになるので、中で条件を変えるか break で抜けます。

int n = 0;
while (n < 3) {
    System.out.println(n);
    n++;
}

【練習1】if で点数に応じた表示

変数 int score = 72; に対して、次のように表示するコードを書いてください。

  • 80 以上 → 「合格（優）」
  • 60 以上 80 未満 → 「合格（良）」
  • 60 未満 → 「不合格」
ヒント

if (score >= 80) { ... } else if (score >= 60) { ... } else { ... } の順で書きます。

解答例 
int score = 72;
if (score >= 80) {
    System.out.println("合格（優）");
} else if (score >= 60) {
    System.out.println("合格（良）");
} else {
    System.out.println("不合格");
}

【練習2】switch で曜日表示

int week = 5; のとき、1→月、2→火、…、7→日 と表示する switch 文を書いてください。1〜7 以外は「不正な値」と表示してください。

ヒント

switch (week) { case 1: ... break; case 2: ... break; ... default: ... break; } のように書きます。

解答例 
int week = 5;
switch (week) {
    case 1: System.out.println("月"); break;
    case 2: System.out.println("火"); break;
    case 3: System.out.println("水"); break;
    case 4: System.out.println("木"); break;
    case 5: System.out.println("金"); break;
    case 6: System.out.println("土"); break;
    case 7: System.out.println("日"); break;
    default: System.out.println("不正な値"); break;
}

【練習3】for で 1 から 5 の合計

for 文を使って 1 から 5 までの整数の合計を求め、System.out.println で表示してください。結果は 15 になります。

ヒント

int sum = 0; を用意し、for (int i = 1; i <= 5; i++) { sum = sum + i; } のあと println(sum); で表示します。

解答例 
int sum = 0;
for (int i = 1; i <= 5; i++) {
    sum = sum + i;
}
System.out.println(sum);  // 15

【発展】while で 0 になるまで 1 引く

int count = 3; から始めて、while で「表示してから 1 引く」を繰り返し、0 になったら終了するプログラムを書いてください。表示は 3, 2, 1 の順で出ればよいです。

ヒント

while (count > 0) { System.out.println(count); count--; } のように書けます。

解答例 
int count = 3;
while (count > 0) {
    System.out.println(count);
    count--;
}

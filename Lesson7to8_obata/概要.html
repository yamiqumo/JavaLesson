【概要】 特別コンテンツ ～boolean・制御構文・クラスと継承～

特別コンテンツ：boolean・制御構文・クラスと継承（約2時間30分）

1. このコンテンツの流れ

次の3パートで、約2時間30分を想定しています。

順番 内容 目安時間
1 boolean の理解を確認 — 真偽値・条件式の確認問題 約20〜25分
2 制御構文 — if / switch / for / while のテキスト＋練習問題 約45〜50分
3 クラスの基礎・コンストラクタ・継承 — テキスト＋練習問題 約70〜80分

2. 前提知識

次の内容を一通り学んでいることを前提にしています。

  • Java の基本構造（クラス・main メソッド）
  • 変数と型（int・String など）
  • 演算子（四則演算・代入）
  • 関係演算子（==・<・> など）の概要

3. 対象ファイル一覧

  • 01_booleanの理解を確認.html — boolean の確認問題
  • 02_制御構文_テキストと練習.html — 制御構文の解説と練習問題
  • 03_クラスの基礎とコンストラクタと継承_テキストと練習.html — クラス・コンストラクタ・継承の解説と練習問題

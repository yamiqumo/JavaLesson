Java研修 練習問題集

基本問題（クラスとオブジェクト）

【問題1】動物クラスを3段階で拡張していく
動物を表す Animal クラスを、次の3段階で少しずつ拡張してください。

フィールド／メソッド一覧（Animal）
区分 名前 型 説明 登場する段階
フィールド leg int 足の数 ステップ1〜3
フィールド name String 動物の名前 ステップ1〜3
メソッド callLegs() void 足の本数を表示する ステップ1〜3
メソッド showInfo() void 動物の情報をまとめて表示する ステップ3
ステップ1： クラスの骨組みを作成する
・Animal クラスを作成し、次のフィールドとメソッドの「宣言」だけを書く。
　- フィールド：int leg（足の数）、String name（名前）
　- メソッド：void callLegs()（処理はまだ空でよい）
ステップ1 ヒント

・まずは「クラスの中身（処理）」は考えず、どんな情報（フィールド）とどんな操作（メソッド）を持つかだけ決めます。
・メソッドは 戻り値の型 メソッド名() の形で宣言します。
・この段階では main メソッドはまだ作らなくて構いません。

ステップ1 解答例 
public class Animal {

  // 足の数
  int leg;

  // 名前
  String name;

  // 足の本数を表示するメソッド（中身はまだ空でOK）
  void callLegs() {
      // ここに処理を書いていく（ステップ2で実装）
  }
}

ステップ2： メソッドの中身を追加する
・callLegs() の中で「〇〇の足は△本です。」と表示する。
ステップ2 ヒント

・callLegs() の中では、name と leg を使って文字列を作り、System.out.println で表示します。
・まずは決め打ちの値（例：name = "ポチ" など）で試してみてから、後で自由に変えられるようにしても構いません。

ステップ2 解答例 
public class Animal {

  int leg;
  String name;

  // 足の本数を表示するメソッド
  void callLegs() {
      System.out.println(name + "の足は" + leg + "本です。");
  }
}

ステップ3： 情報表示メソッドと main メソッドからの利用（コンストラクタは使わず、フィールドに直接代入する）
・showInfo() メソッドを追加し、名前と足の数をまとめて表示する。
・別クラス AnimalMain を作成し、その中の main メソッドで複数の Animal を生成し、フィールドに直接値を代入してから callLegs() と showInfo() を呼び出して確認する。
ステップ3 ヒント

・showInfo() では、System.out.println を複数行書いて、見やすい形で情報を表示します。
・main メソッドは public static void main(String[] args) という形で書きます。
・Animal と AnimalMain は別のファイルに分けても、同じファイルに2クラス書いても構いません（学習用）。
・将来的には、Animal を親クラスにしてサブクラスを作る練習につなげられます。

ステップ3 解答例 
// ---------- 「Animal.class」
public class Animal {

  int leg;
  String name;

  void callLegs() {
      System.out.println(name + "の足は" + leg + "本です。");
  }

  // 情報表示用メソッド
  void showInfo() {
      System.out.println("名前: " + name);
      System.out.println("足の数: " + leg + "本");
  }
}

// ---------- 「AnimalMain.class」
public class AnimalMain {

  public static void main(String[] args) {
      Animal dog = new Animal();
      dog.leg = 4;
      dog.name = "ポチ";

      Animal cat = new Animal();
      cat.leg = 2;
      cat.name = "タマ";

      dog.callLegs();
      cat.callLegs();

      dog.showInfo();
      cat.showInfo();

      Animal unknown = new Animal();
      unknown.leg = 0;
      unknown.name = "名前なし";
      unknown.callLegs();
      unknown.showInfo();
  }
}




【問題2】社員クラスを3段階で拡張していく
社員を表す Employee クラスを、次の3段階で少しずつ拡張してください。

フィールド／メソッド一覧（Employee）
区分 名前 型 説明 登場する段階
フィールド name String 社員の氏名 ステップ1〜3
フィールド id int 社員番号 ステップ1〜3
フィールド baseSalary int 基本給（月額・円） ステップ2〜3
フィールド overtimeHours int 残業時間（月・時間） ステップ2〜3
メソッド showInfo() void 社員情報を表示する ステップ1〜3（内容を拡張）
メソッド calcTotalSalary() int 支給額（基本給＋残業代）を計算して返す ステップ2〜3
ステップ1： 氏名と社員番号だけを扱うクラス
・Employee クラスを作成し、次のフィールドとメソッドを宣言・実装する。
　- フィールド：String name（氏名）、int id（社員番号）
　- メソッド：void showInfo()（名前と社員番号だけを表示）
ステップ1 ヒント

・まずは「社員とは何か？」を、名前と番号だけで表すシンプルなクラスとして考えます。
・showInfo() では、System.out.println を2行書いて、name と id を表示します。
・この段階では給料に関するフィールドはまだ登場させません。

ステップ1 解答例 
public class Employee {

  String name;
  int id;

  // 社員情報を表示するメソッド
  void showInfo() {
      System.out.println("名前: " + name);
      System.out.println("社員番号: " + id);
  }
}

ステップ2： 給料に関するフィールドと計算メソッドの追加
・フィールドを追加する。
　- int baseSalary（基本給）
　- int overtimeHours（残業時間）
・int calcTotalSalary() メソッドを追加し、
　- 残業代 ＝ overtimeHours * 1000 円
　- 支給額 ＝ baseSalary + 残業代
　として計算し、その結果を返す。
ステップ2 ヒント

・計算用メソッド calcTotalSalary() は、「計算だけして結果を返す」役割にすると、後で拡張しやすくなります。
・計算に使う数値（残業時間・時給）は、すべて int で扱って構いません。

ステップ2 解答例 
public class Employee {

  String name;
  int id;
  int baseSalary;
  int overtimeHours;

  // 支給額を計算して返すメソッド
  int calcTotalSalary() {
      int overtimePay = overtimeHours * 1000;
      int total = baseSalary + overtimePay;
      return total;
  }

  void showInfo() {
      System.out.println("名前: " + name);
      System.out.println("社員番号: " + id);
  }
}

ステップ3： 情報表示の拡張と main メソッドからの利用（コンストラクタは使わず、フィールドに直接代入する）
・showInfo() を拡張し、基本給・残業時間・支給額も表示するようにする。
・別クラス EmployeeMain を作成し、その中の main メソッドで社員を生成し、showInfo() を呼び出す。
ステップ3 ヒント

・showInfo() の中から calcTotalSalary() を呼び出して、支給額を取得します。
・main メソッドでは、Employee emp1 = new Employee(); のようにインスタンスを生成し、name や id などのフィールドに直接値を代入します。
・ここまでできると、将来的に Employee を抽象クラスにし、RegularEmployee や Manager クラスに継承させる練習がしやすくなります。

ステップ3 解答例 
// ---------- 「Employee.class」
public class Employee {

  String name;
  int id;
  int baseSalary;
  int overtimeHours;

  int calcTotalSalary() {
      int overtimePay = overtimeHours * 1000;
      int total = baseSalary + overtimePay;
      return total;
  }

  void showInfo() {
      System.out.println("名前: " + name);
      System.out.println("社員番号: " + id);
      System.out.println("基本給: " + baseSalary + "円");
      System.out.println("残業時間: " + overtimeHours + "時間");

      int total = calcTotalSalary();
      System.out.println("支給額: " + total + "円");
  }
}

// ---------- 「EmployeeMain.class」
public class EmployeeMain {

  public static void main(String[] args) {
      Employee emp1 = new Employee();
      emp1.name = "山田太郎";
      emp1.id = 1001;
      emp1.baseSalary = 200000;
      emp1.overtimeHours = 10;
      emp1.showInfo();
  }
}




発展問題（社員クラスのロジック）

【問題3】社員クラスの給料ロジックを発展させる
【問題2】で作成した Employee クラスをもとに、繰り返し（for文）と条件分岐（if文）を使ったロジックを追加してください。

仕様（発展）
1. Employee クラスに、次のメソッドを追加します。
　・int calcOvertimePayDetail()
　　- 残業時間を1時間ずつカウントしながら、for文で残業代を計算する。
　　- 1〜20時間目までは 1時間あたり1000円。
　　- 21時間目以降は 1時間あたり1500円（割増）。
　　- if文を使って、「20時間以内かどうか」で金額を切り替える。
　・int calcTotalSalaryDetail()
　　- 上記の calcOvertimePayDetail() を使って、
　　　基本給 + 詳細な残業代 を計算して返す。
2. 別クラス EmployeeAdvancedMain を作成し、
　・配列 Employee[] employees に3人分の社員を用意する。
　・for文で配列をまわし、各社員の情報と詳細な支給額を表示する。
　・同じループの中で最大支給額の社員を調べ、最後に「最も支給額が高い社員」を表示する（if文で比較）。
発展問題 ヒント

・残業代の計算では、for (int i = 1; i <= overtimeHours; i++) のように1時間ずつカウントしていきます。
　その中で、if (i <= 20) なら1000円、それ以外は1500円を加算します。
・最大支給額の社員を探すときは、
　「今までの最大値」と「今回の社員の支給額」を if文で比較して、大きい方を保存していきます。
・この発展問題は、Employee クラスを基底クラスとして、
　「ロジックだけを少しずつ複雑にしていく」練習としても使えます。

発展問題 解答例 
// ---------- 「Employee.class（発展）」
public class Employee {

  String name;
  int id;
  int baseSalary;
  int overtimeHours;

  // 発展版：1時間ずつfor文で残業代を計算する
  int calcOvertimePayDetail() {
      int overtimePay = 0;

      for (int i = 1; i <= overtimeHours; i++) {
          if (i <= 20) {
              overtimePay = overtimePay + 1000;
          } else {
              overtimePay = overtimePay + 1500;
          }
      }

      return overtimePay;
  }

  // 発展版：詳細な残業代を使って支給額を計算
  int calcTotalSalaryDetail() {
      int overtimePay = calcOvertimePayDetail();
      int total = baseSalary + overtimePay;
      return total;
  }

  void showInfo() {
      System.out.println("名前: " + name);
      System.out.println("社員番号: " + id);
      System.out.println("基本給: " + baseSalary + "円");
      System.out.println("残業時間: " + overtimeHours + "時間");

      int overtimePay = calcOvertimePayDetail();
      int total = calcTotalSalaryDetail();

      System.out.println("詳細な残業代: " + overtimePay + "円");
      System.out.println("支給額(発展版): " + total + "円");
      System.out.println();
  }
}

// ---------- 「EmployeeAdvancedMain.class」
public class EmployeeAdvancedMain {

  public static void main(String[] args) {
      Employee[] employees = new Employee[3];

      employees[0] = new Employee();
      employees[0].name = "山田太郎";
      employees[0].id = 1001;
      employees[0].baseSalary = 200000;
      employees[0].overtimeHours = 10;

      employees[1] = new Employee();
      employees[1].name = "佐藤花子";
      employees[1].id = 1002;
      employees[1].baseSalary = 220000;
      employees[1].overtimeHours = 25;

      employees[2] = new Employee();
      employees[2].name = "鈴木一郎";
      employees[2].id = 1003;
      employees[2].baseSalary = 180000;
      employees[2].overtimeHours = 5;

      int maxTotal = 0;
      Employee maxEmployee = null;

      for (int i = 0; i < employees.length; i++) {
          Employee emp = employees[i];

          emp.showInfo();

          int total = emp.calcTotalSalaryDetail();

          if (total > maxTotal) {
              maxTotal = total;
              maxEmployee = emp;
          }
      }

      System.out.println("最も支給額が高い社員: " + maxEmployee.name);
      System.out.println("支給額(発展版): " + maxTotal + "円");
  }
}

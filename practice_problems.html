    }
}

【問題2】 1からnまでの整数の合計を計算して表示してください（n=5とします）。
ヒント

合計を保持する変数を用意し、for文で1からnまで繰り返しながら、各数値を合計に加算していきます。最後に合計を表示します。

解答１ 
public static void main(String[] args) {
    // 1から5までの整数の合計を計算して表示（メインメソッドのみ）
    int n = 5;
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum = sum + i;
    }
    System.out.println("合計: " + sum);
}
解答２ 
public static int calculateSum(int n) {
    // 1からnまでの整数の合計を計算する
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum = sum + i;
    }
    int result = sum;
    return result;
}

public static void main(String[] args) {
    // 1から5までの合計を計算して表示
    int total = calculateSum(5);
    System.out.println("合計: " + total);
}

複合・総合問題

【問題25】 1からnまでの整数のうち、偶数の合計と奇数の合計をそれぞれ計算し、大きい方の値を返すメソッドを作成してください。
メソッド名：getLargerSum
引数：int n
戻り値：int
ヒント

for文で1からnまで繰り返し、if文で偶数（i % 2 == 0）と奇数に分けてそれぞれ合計を計算します。その後、if文で大きい方の値を判定し、変数に代入してからreturnします。mainメソッドで作成してもいいですよ。

解答１ 
public static void main(String[] args) {
    // 1から10までの偶数の合計と奇数の合計の大きい方を計算して表示（メインメソッドのみ）
    int n = 10;
    int evenSum = 0;
    int oddSum = 0;
    for (int i = 1; i <= n; i++) {
        if (i % 2 == 0) {
            evenSum = evenSum + i;
        } else {
            oddSum = oddSum + i;
        }
    }
    int largerSum;
    if (evenSum > oddSum) {
        largerSum = evenSum;
    } else {
        largerSum = oddSum;
    }
    System.out.println("大きい方の合計: " + largerSum);
}
解答２ 
public static int getLargerSum(int n) {
    // 1からnまでの偶数の合計と奇数の合計の大きい方を返す
    int evenSum = 0;
    int oddSum = 0;
    for (int i = 1; i <= n; i++) {
        if (i % 2 == 0) {
            evenSum = evenSum + i;
        } else {
            oddSum = oddSum + i;
        }
    }
    int largerSum;
    if (evenSum > oddSum) {
        largerSum = evenSum;
    } else {
        largerSum = oddSum;
    }
    int result = largerSum;
    return result;
}

public static void main(String[] args) {
    // 偶数の合計と奇数の合計の大きい方を計算して表示
    int larger = getLargerSum(10);
    System.out.println("大きい方の合計: " + larger);
}

【問題27】 文字列を引数として受け取り、その文字列が「開始」の場合は「処理開始」と表示してtrueを返し、「終了」の場合は「処理終了」と表示してfalseを返し、それ以外の場合は「無効」と表示してfalseを返すメソッドを作成してください。また、mainメソッドでwhile文を使って、「開始」が返されるまで繰り返し処理を実行してください。
メソッド名：processCommand
引数：String command
戻り値：boolean
ヒント

if-else if-else文で文字列を判定し、戻り値をboolean変数に代入してからreturnします。mainメソッドでは、boolean変数を用意し、while文で「開始」が返されるまで（!isStarted）繰り返します。文字列の比較には必ず.equals()を使用します。mainメソッドで作成してもいいですよ。

解答１ 
public static void main(String[] args) {
    // コマンド処理を繰り返し実行（開始が返されるまで、メインメソッドのみ）
    boolean isStarted = false;
    int count = 0;
    while (!isStarted) {
        String command;
        if (count == 0) {
            command = "待機";
        } else if (count == 1) {
            command = "準備";
        } else {
            command = "開始";
        }
        boolean shouldContinue;
        if (command.equals("開始")) {
            System.out.println("処理開始");
            shouldContinue = true;
        } else if (command.equals("終了")) {
            System.out.println("処理終了");
            shouldContinue = false;
        } else {
            System.out.println("無効");
            shouldContinue = false;
        }
        boolean result = shouldContinue;
        isStarted = result;
        count++;
    }
}
解答２

【問題28】 整数nを引数として受け取り、1からnまでの整数を順番に表示するが、素数（1とその数自身以外で割り切れない数）の場合は「[素数]」という文字列を付けて表示するメソッドを作成してください。
メソッド名：printNumbersWithPrimes
引数：int n
戻り値：void
ヒント

for文で1からnまで繰り返します。1の場合はそのまま表示し、2以上の場合は、2からi-1までで割り切れるかどうかを内側のfor文で判定します。割り切れる数があれば素数ではないので、boolean変数で管理します。素数判定の結果に応じて表示を分岐します。mainメソッドで作成してもいいですよ。

解答１ 
public static void main(String[] args) {
    // 1から10までの数値を表示し、素数の場合は[素数]を付ける（メインメソッドのみ）
    int n = 10;
    for (int i = 1; i <= n; i++) {
        if (i == 1) {
            System.out.println(i);
        } else {
            boolean isPrime = true;
            for (int j = 2; j < i; j++) {
                if (i % j == 0) {
                    isPrime = false;
                    break;
                }
            }
            if (isPrime) {
                System.out.println(i + " [素数]");
            } else {
                System.out.println(i);
            }
        }
    }
}
